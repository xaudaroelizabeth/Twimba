# 🐦 Twimba

A fun Twitter-style social media app built with HTML, CSS, and JavaScript.

---

## ✨ About the Project

Twimba is a mini social media app where users can post tweets, like posts, retweet, comment, and delete tweets (with restrictions).

It also saves data in the browser so your posts don’t disappear when you refresh!

---

## 🚀 Features

💬 Post new tweets  
❤️ Like and unlike tweets  
🔁 Retweet posts  
💭 Add comments/replies  
🗑️ Delete your own tweets  
💾 Save data with localStorage  
🆔 Unique tweet IDs with UUID  

---

## 🧠 How It Works

- Tweets are stored in a JavaScript array
- Each tweet has likes, retweets, and replies
- Clicking icons updates the data in real time
- Everything re-renders automatically
- Data is saved in the browser using localStorage

---

## 🛠️ Built With

- HTML5  
- CSS3  
- JavaScript (ES6 Modules)  
- Font Awesome Icons
